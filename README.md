# phucbui.me Front-end
## Static Version of phucbui.me
* GulpJS
* SASS
* Pug

### Demo
https://phucbui.me

### Run
`gulp serve`

App will be running at `http://localhost:3000`

### Build
`gulp` or `gulp build`
App will be built at `/dist`
